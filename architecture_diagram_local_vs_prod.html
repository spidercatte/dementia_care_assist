Local vs Production Decoupled Architectures

Comparing the offline multi-container configuration with the cloud serverless/managed Google Cloud stack.

Local Development (Docker Compose)

The Web App runs all agent logic locally. A separate ADK Agent Playground can be run locally for testing, connecting to FastMCP over SSE.

Production Environment (Google Cloud)

Decoupled 5-service cloud stack. The Backend API invokes the Agent (Reasoning Engine), which queries the Datastore natively and pulls DB records via FastMCP SSE.
