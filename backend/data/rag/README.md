# DementiaCare Coach RAG Knowledge Base

This folder contains Markdown guideline files designed for Retrieval-Augmented Generation (RAG).

## Recommended Retrieval Strategy

Use multiple focused Markdown files instead of one large document.

Each file includes:
- YAML frontmatter metadata
- Scenario triggers
- Risk level
- Escalation rules
- Caregiver scripts
- Things to avoid
- Personalization notes
- Optional Mermaid decision flow

## Suggested RAG Output Format

When generating coaching advice, the assistant should return:

1. What may be happening
2. What the caregiver should do now
3. Exact words the caregiver can say
4. What to avoid
5. Patient-specific warnings
6. Whether escalation is needed

## Important Safety Note

This knowledge base is for caregiver coaching support and does not replace clinical judgment, emergency services, or professional medical advice.
